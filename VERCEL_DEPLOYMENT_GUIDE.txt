# GAPP - Grocery App with Supabase & Vercel

## Quick Start for Vercel Deployment

### 1. Install Git (if not already installed)
Download from: https://git-scm.com/download/win

### 2. Clone or Initialize Repository
```bash
cd "c:\Users\DYNAMIC-COMPUTER\OneDrive\Documents\GAPP"

# Initialize git
git config --global user.name "Ali Hamza Khan"
git config --global user.email "your-email@example.com"
git init

# Add all files
git add .

# Create initial commit
git commit -m "Initial commit: Grocery app with Supabase integration and order management"

# Add remote repository
git remote add origin https://github.com/alihamza-khan/gapp.git

# Push to main branch
git branch -M main
git push -u origin main
```

### 3. Set Environment Variables in Vercel

In Vercel Dashboard → Project Settings → Environment Variables

Add these variables:
- **NEXT_PUBLIC_SUPABASE_URL**: https://rltpcldfgmarxfbmflgl.supabase.co
- **NEXT_PUBLIC_SUPABASE_ANON_KEY**: (copy from .env.local)

### 4. Deploy to Vercel

1. Go to https://vercel.com
2. Click "New Project"
3. Connect your GitHub repository
4. Select this project from the list
5. Configure environment variables (see step 3)
6. Click "Deploy"

### 5. Database Setup

Your Supabase database schema is already in sql/groceryapp_schema.sql

To seed data on first API call:
- Visit your-domain.vercel.app
- Add products to cart
- Click "Place Order"
- The API will automatically seed products on demand

## Project Structure

- **src/app/**: Next.js App Router pages
  - **api/orders**: Order management API
  - **api/seed**: Database seeding API
  - **cart**: Shopping cart page
  - **products**: Product listing page
  
- **src/components**: React components
- **src/hooks**: Custom hooks (useCart)
- **src/lib**: Utilities and configurations
- **sql**: Database schema

## Troubleshooting

**Products not showing after order:**
- API automatically seeds products on first order
- Check Supabase dashboard for seeded data

**Image 404 errors:**
- Pexels CDN URLs are configured in next.config.js
- Some URLs may fail if images were deleted - app handles gracefully

**Order not saving:**
- Ensure Supabase credentials are correct
- Check browser console for specific errors
- Verify network request in DevTools

## Support

For production deployment:
1. Update Supabase URL and keys for production
2. Enable RLS (Row Level Security) in Supabase
3. Configure CORS properly
4. Set up monitoring with Vercel Analytics
